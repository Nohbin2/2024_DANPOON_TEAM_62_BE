🌱 여름지기 커뮤니티 백엔드 (DANPOON TEAM 62)
📌 프로젝트 소개

여름지기는 새싹 여름지기부터 기존 여름지기까지,
한 해의 경험을 공유하고 함께 성장할 수 있도록 기획된 커뮤니티 서비스입니다.
본 레포지토리는 해당 서비스의 백엔드 서버 구현 코드를 담고 있습니다.

본 프로젝트는 9oormthon Univ 2024 (단풍톤) 팀 프로젝트로 진행되었습니다.

🛠 기술 스택

Language: Java

Framework: Spring Boot

Build Tool: Gradle

Architecture: RESTful API, 계층 분리 구조 (Controller / Service / Repository)

✨ 주요 기능

커뮤니티 서비스 백엔드 구현

커뮤니티 도메인 중심의 REST API 설계

게시글 및 사용자 관련 데이터 처리

데이터 CRUD 처리

커뮤니티 서비스에 필요한 핵심 데이터 생성·조회·수정·삭제

확장 가능한 서버 구조

추후 프론트엔드 및 외부 서비스 연동을 고려한 API 설계

👥 역할 분담

본 레포지토리는 팀 프로젝트의 백엔드 코드 저장소입니다.

팀원 간 역할을 분담하여 백엔드 서버 개발을 진행했습니다.

📂 프로젝트 구조
├─ gradle/
├─ src/
│  └─ main/
│     └─ java/
├─ build.gradle
├─ settings.gradle
└─ gradlew

🚀 프로젝트 목적

Spring Boot 기반 백엔드 서버 구조 설계 경험

팀 프로젝트를 통한 협업 개발 경험

커뮤니티 서비스의 기본 도메인 및 API 흐름 이해

📌 참고 사항

본 프로젝트는 해커톤/학습 목적의 프로젝트로,
실제 서비스 운영 단계까지는 진행하지 않았습니다.

커뮤니티 서비스의 핵심 구조 구현에 초점을 맞춘 백엔드 레포지토리입니다.
